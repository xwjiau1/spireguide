# SpireGuide 使用说明

## 功能概览

尖塔智囊 (SpireGuide) 是一款面向《杀戮尖塔》(Slay the Spire) 玩家的 AI 攻略助手，支持：

- 🤖 **AI截图问答** — 上传游戏截图，AI识别当前状态并给出策略建议
- 📚 **卡牌数据库** — 浏览、搜索、筛选 StS1/StS2 全部卡牌
- 👹 **敌人数据库** — 查看每个敌人的意图模式和应对策略
- 💎 **遗物数据库** — 浏览全部遗物效果
- 📸 **对局记录** — 记录关键节点截图和AI策略，回溯对局历程

## 快速开始

### 1. 启动服务

```bash
cd 06-交付/build
bash start.sh
```

浏览器访问 `http://localhost:3001`

### 2. AI截图问答（核心功能）

**步骤：**
1. 进入首页「AI截图助手」
2. 拖拽或粘贴游戏截图到上传区域
3. 等待AI识别当前场景（约1-2秒）
4. **检查识别结果** — 确认HP、能量、手牌、敌人等信息是否正确
5. 如有错误，直接点击数值修改
6. 点击「确认识别结果」
7. AI生成策略建议（含出牌优先级、敌人应对要点）
8. 点击「保存到对局记录」可将本次问答存档

**重要：** 策略回答底部标注 **【仅供参考，请结合实际判断】**，AI建议不能替代玩家判断。

### 3. 卡牌查询

- 顶部导航进入「卡牌数据库」
- 搜索框支持中英文模糊搜索（如"打击"或"Strike"）
- 筛选器支持：游戏版本(StS1/StS2)、角色、类型(攻击/技能/能力)、稀有度
- 点击卡牌卡片查看完整详情（基础效果、升级后效果、关键词）

### 4. 敌人查询

- 进入「敌人数据库」
- 可按章节(Act 1-4)、类型(普通/精英/BOSS)筛选
- 查看敌人意图模式、通用对策、各角色专属对策

### 5. 对局记录

- 进入「对局记录」
- 点击「新建对局」输入角色、进阶等级、种子
- 对局中可随时截图并保存到当前对局时间线
- 对局结束后可标记结局（胜利/失败/放弃）

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + V` | 首页粘贴截图 |
| `Enter` | 搜索框触发搜索 |
| `ESC` | 关闭弹窗/详情面板 |

## 常见问题

**Q: 为什么AI策略有时不准确？**
A: 当前为MVP阶段，AI使用启发式策略+数据库增强。接入真实Kimi API后准确率将大幅提升。请始终结合自己的判断。

**Q: StS2内容完整吗？**
A: StS2当前为Early Access阶段，数据库收录了EA已知内容（缚魂者、摄政王卡牌等）。正式发售后将更新完整数据。

**Q: 数据会丢失吗？**
A: 所有数据存储在本地 SQLite 数据库，只要不删除 `data/spireguide.db` 文件，数据不会丢失。

**Q: 如何接入真实Kimi API？**
A: 部署时设置环境变量 `KIMI_API_KEY=your_key`，并修改 `src/backend/src/routes/ai.ts` 中的 Mock 实现为真实 API 调用。
